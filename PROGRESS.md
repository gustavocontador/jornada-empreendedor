### Por que FastAPI?
- Performance superior (async)
- Documentação automática
- Type hints nativos
- Validação Pydantic

### Por que Next.js 14?
- Server components
- App router
- Performance excelente
- SEO friendly

### Por que PostgreSQL?
- JSONB para flexibilidade
- Reliability
- Full-text search nativo
- Indexes poderosos

### Por que WeasyPrint?
- CSS support completo
- HTML → PDF perfeito
- Controle total de layout
